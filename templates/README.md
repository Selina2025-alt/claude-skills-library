# Claude Code 模板库

这里存放全局通用的项目说明书模板。

## 📋 可用模板

### 通用_CLAUDE.md

适用于所有项目的全局配置模板，包含：

- **项目说明**：项目描述和技术栈
- **常用命令**：项目相关命令
- **核心行为规范**：中文回答、代码注释要求
- **Git 版本管理规范**：
  - 存档流程（提交到 Git）
  - 回退流程（撤销更改）
  - 历史查询

## 📝 使用方法

### 新项目使用模板

```bash
# 复制模板到新项目
cp templates/通用_CLAUDE.md 你的项目路径/CLAUDE.md

# 然后根据项目实际情况修改模板内容
```

### 模板更新说明

当更新全局模板时：
1. 修改 `templates/通用_CLAUDE.md`
2. 提交到 Git 仓库
3. 在现有项目中运行 `cp` 命令同步更新
