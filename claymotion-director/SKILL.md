# ClayMotion Director (黏土定格动画导演)

## 触发方式

当用户说类似以下内容时触发：
- "做个黏土动画视频 / 定格动画"
- "生成毛毡风/黏土风分镜提示词"
- "把这段文案转成动画分镜"
- "/claymotion" "/黏土动画" "/clay"

---

## 核心功能 🔥

### 一键式工作流
1. **需求澄清** - 询问风格、画幅等偏好
2. **文案清晰化** - 自动整理杂乱文案，补充开场和总结
3. **智能分镜拆解** - 根据内容自动分配镜头和时长
4. **完整提示词生成** - 为每个分镜生成**完整详细**的提示词（不做任何简化）

## 调度规则

**如何判断当前阶段：**
1. 用户刚提供文案 → **阶段 0**（先询问风格和画幅）
2. 已确认风格，还没清晰化文案 → 阶段1（清晰化文案）
3. 文案已清晰化，但还没拆分镜 → 阶段2（自动分镜拆解）
4. 分镜脚本已完成 → 阶段3（生成完整提示词）

**每个阶段开始时：**
- 告诉用户当前阶段与本阶段输出物
- 读取对应阶段文件并按步骤执行

## 风格系统

本技能支持两种核心黏土动画风格：

### Style A: Tech-Polymer (光面黏土风)
- **质感**: 类似 Polymer Clay 的光面质感，指纹可见
- **关键词**: smooth polymer clay, glossy surface, vibrant colors, miniature set
- **适合**: 技术产品、API、架构、数据分析
- **色彩**: 深蓝/霓虹绿/紫红/银灰

### Style B: Cozy-Felt (暖心羊毛毡风)
- **质感**: 类似 Needle Felting 的毛绒质感，纤维蓬松
- **关键词**: needle felting, wool texture, fuzzy surface, warm lighting
- **适合**: 用户故事、情感共鸣、教育内容
- **色彩**: 米黄/粉橙/草绿/天蓝

## 需求澄清（阶段 0 必做）

收到用户文案后，**必须先暂停**，使用 `AskUserQuestion` 工具询问：

**问题 1：选择动画风格**
- **A. 光面黏土风**: 表面光滑细腻，类似精致盲盒玩具，色彩明亮。适合表现科技感、高效、现代氛围。
- **B. 暖心毛毡风**: 表面毛茸茸，有纤维质感，边缘柔和。适合表现温暖、情感共鸣、手工氛围。

**问题 2：视频画幅**
- **16:9 横屏** (推荐，适合 YouTube/视频号)
- **9:16 竖屏** (适合抖音/快手/短视频)
- **1:1 方形** (适合社交媒体)

## 快速使用（给用户的最小输入）

用户只需要提供：
1. **一大段文案**（可以是杂乱的、不完整的）
2. **风格偏好**（通过 AskUserQuestion 选择）

**技能会自动完成**：
- ✅ 清晰化文案（补充开场、理顺逻辑、添加总结）
- ✅ 拆解分镜（自动分配时长、设计动作、规划镜头）
- ✅ 生成提示词（每个分镜都有完整详细的提示词）

## 输出规范（必须遵守）

### 文案清晰化规范
- 必须包含：开场引入 + 核心关键点 + 总结升华
- 逻辑清晰：问题→解决→价值的完整结构
- 语言通俗：将专业术语转化为视觉化表达

### 分镜拆解规范
- 每个分镜时长控制在 3-8 秒
- 每个分镜只有一个核心动作
- 动作设计遵循"起始→核心→结束"三段式
- 同一角色在不同场景中保持造型一致

### 提示词生成规范
**每个场景的提示词必须包含**：
1. **风格材质**: 完整描述（> 50 字符）
2. **场景环境**: 详细描述
3. **角色形象**: 造型/颜色/状态
4. **核心动作**: 三段式描述
5. **光影描述**: 光源类型+色调
6. **镜头语言**: 类型 + 运动
7. **灵活动效**: 鼓励动态镜头，不限制技术参数

**提示词格式要求**：
- **Midjourney 提示词**: 英文，可直接复制，包含风格关键词 + 场景 + 动作 + 光影 + --ar 参数
- **视频动态指导**: 中文，描述核心动作、镜头运动、转场方式
