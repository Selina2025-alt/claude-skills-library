## 核心理念：视觉叙事 🎬

**画面讲故事，而非角色口播。**

本技能的核心是将文案转化为**丰富的视觉叙事**，而不是单一角色面对镜头的口播模式。

### 视觉叙事原则
- **画面即内容**：用场景、物品、动态来可视化文案含义
- **角色是元素**：角色可以出现也可以不出现，服务于内容表达
- **镜头多变**：正面、侧面、背面、俯视、特写，根据内容选择最佳角度
- **动态丰富**：物品动态、场景变化、镜头运动，不局限于角色动作

### 核心公式
```
文案内容 → 视觉隐喻 → 画面构成 → 镜头角度 → 动态设计 → 提示词
```

## 风格系统

本技能支持六种核心视觉风格：

### Style A: Tech-Polymer (光面黏土风)
- **质感**: 类似 Polymer Clay 的光面质感，指纹可见
- **关键词**: smooth polymer clay, glossy surface, vibrant colors, miniature set
- **适合**: 技术产品、API、架构、数据分析
- **色彩**: 深蓝/霓虹绿/紫红/银灰

### Style B: Cozy-Felt (暖心羊毛毡风)
- **质感**: 类似 Needle Felting 的毛绒质感，纤维蓬松
- **关键词**: needle felting, wool texture, fuzzy surface, warm lighting
- **适合**: 用户故事、情感共鸣、教育内容
- **色彩**: 米黄/粉橙/草绿/天蓝

### Style C: Pixel-Nostalgia (像素怀旧风)
- **质感**: 方块化像素颗粒，清晰可见的像素格子，阶梯状锯齿边缘
- **关键词**: pixel art, 16-bit retro, visible pixel grid, dithering gradients
- **适合**: 游戏/怀旧内容、生活日常、城市旅行、文化主题
- **色彩**: 风格多样，从暖色到明亮到灰蓝到红金

### Style D: Neon-Lofi (霓虹慢波风)
- **质感**: 细腻数字插画，线条精致，细节极其丰富密集
- **关键词**: neon purple magenta, lo-fi anime, night atmosphere, dense details
- **适合**: 游戏/数码内容、城市夜景、Lo-Fi音乐、赛博朋克
- **色彩**: 紫粉霓虹/洋红/电紫/深蓝

### Style E: Watercolor-Dream (水彩梦境风)
- **质感**: 水彩纸纹理可见，颜料晕染自然渗透，边缘柔和模糊
- **关键词**: watercolor painting, paper texture, soft diffused edges, transparent layers
- **适合**: 旅行风光、生活故事、儿童绘本、文艺诗意
- **色彩**: 清透淡雅，自然色调

### Style F: Fabric-Craft (布艺温暖风)
- **质感**: 粗麻布/毛毡/拼布材质，可见粗针脚和缝线细节，磨损边缘和松散线头，布料自然褶皱和堆叠，纽扣眼睛，绣花表情
- **关键词**: stop-motion fabric craft animation, coarse burlap, woven jute, felt texture, fraying edges, loose threads, visible thick stitching, patchwork design, mismatched button eyes, embroidered features, handmade plushie aesthetic
- **摄影特征**: 浅景深/背景虚化 (shallow depth of field, bokeh background), 微距镜头 (macro lens), 定格动画特有的略微顿挫感 (jerky movement, 12fps feel), 莱卡工作室风格 (Laika Studios style)
- **适合**: 儿童/亲子/教育、生活方式/治愈系、科技产品亲和化表达、温暖叙事、手作感内容
- **色彩**: 暖色调 (warm color palette)，自然布料色系，根据内容主题灵活调整
- **环境**: 复古木桌、温馨工作室、暖光窗边、丁达尔光效（阳光尘埃）

## 需求澄清（阶段 0 必做）

收到用户文案后，**必须先暂停**，使用 `AskUserQuestion` 工具询问：

**问题 1：选择动画风格**
- **A. 光面黏土风**: 表面光滑细腻，类似精致盲盒玩具，色彩明亮。适合科技感、高效、现代氛围。
- **B. 暖心毛毡风**: 表面毛茸茸，有纤维质感，边缘柔和。适合温暖、情感共鸣、手工氛围。
- **C. 像素怀旧风**: 方块化像素颗粒，复古游戏美学。适合怀旧、游戏、生活日常、城市旅行。
- **D. 霓虹慢波风**: 紫粉霓虹色调，Lo-Fi美学，极致细节。适合夜景、数码科技、赛博朋克。
- **E. 水彩梦境风**: 水彩纸纹理，颜料晕染，诗意梦幻。适合旅行风光、生活故事、文艺内容。
- **F. 布艺温暖风**: 软质布艺，可见缝线细节，马卡龙配色。适合儿童/亲子/教育、治愈系、科技亲和化表达。

**问题 2：视频画幅**
- **16:9 横屏** (推荐，适合 YouTube/视频号)
- **9:16 竖屏** (适合抖音/快手/短视频)
- **1:1 方形** (适合社交媒体)

## 输出规范（必须遵守）

### 视觉隐喻设计规范
- 每句文案都要有对应的视觉化表达方式
- 判断是否需要角色出现（角色是可选的）
- 选择最能表达文案含义的视觉隐喻

### 分镜设计规范
- 每个分镜时长控制在 4-8 秒
- 明确画面构成：前景/中景/背景各有什么
- 选择最佳镜头角度：正面/侧面/背面/俯视/特写
- 设计动态效果：物品动态、场景动态、镜头动态

### 提示词生成规范（Veo3 格式）
**每个场景的提示词必须包含**：
1. **风格声明**: Stop-motion claymation + 材质风格
2. **镜头角度**: 正面/侧面/俯视/特写 + 焦距描述
3. **画面构成**: 前景、中景、背景的具体内容
4. **主体描述**: 角色/物品的详细外观（如有角色）
5. **动态设计**: 物品动态 + 场景动态 + 角色动作
6. **镜头运动**: static/dolly/tracking/pan 等
7. **光影氛围**: 光源类型 + 色调 + 情绪

**提示词格式要求**：
- **Veo3 提示词**: 英文，可直接复制使用
- **分镜说明**: 中文，描述画面内容和视觉隐喻
