# 视觉叙事指南 (Visual Storytelling Guide)

本指南提供视觉叙事的核心原则、镜头语言、画面构成和动态设计参考。

---

## 核心原则：画面讲故事

**视觉叙事的本质**：用画面元素的组合和变化来传达信息，而不是依赖文字或角色口述。

### 好的视觉叙事特征

| 特征 | 说明 |
|------|------|
| **即时可读** | 观众在 1-2 秒内理解画面含义 |
| **情感共鸣** | 画面引发观众的情感反应 |
| **记忆深刻** | 独特的视觉形象让人印象深刻 |
| **无需解释** | 不依赖文字说明也能理解 |

### 视觉叙事 vs 口播模式

| 维度 | ❌ 口播模式 | ✅ 视觉叙事 |
|------|-----------|-----------|
| 信息传递 | 角色说出信息 | 画面演示信息 |
| 角色作用 | 主体、信息源 | 画面元素之一 |
| 观众体验 | 被动听讲 | 主动发现 |
| 记忆效果 | 容易遗忘 | 印象深刻 |

## 镜头语言指南

### 镜头角度与情感

| 角度 | 心理效果 | 最佳使用场景 |
|------|---------|------------|
| **正面平视** | 亲密、平等、直接 | 与观众建立连接、情感表达 |
| **俯视** | 掌控、全局、客观 | 展示布局、流程、关系图 |
| **仰视** | 崇敬、权威、力量 | 展示重要人物、权威来源 |
| **侧面** | 动态、叙事、过程 | 展示动作、移动、变化 |
| **背面** | 神秘、展望、带入 | 引导视线、展示目标 |
| **斜角** | 紧张、不安、动态 | 冲突、紧张时刻 |

### 景别与信息量

| 景别 | 信息量 | 情感距离 | 使用场景 |
|------|-------|---------|---------|
| **远景** | 环境为主 | 疏远、客观 | 建立场景、展示全局 |
| **全景** | 人物全身 | 中等 | 展示动作、位置关系 |
| **中景** | 人物膝上 | 适中 | 日常叙事、对话 |
| **近景** | 人物胸上 | 亲近 | 表情、情感 |
| **特写** | 局部细节 | 极近 | 关键物品、微表情 |

### 镜头运动与节奏

| 运动 | 节奏感 | 情感效果 | 使用场景 |
|------|-------|---------|---------|
| **静止** | 稳定 | 专注、庄重 | 重要时刻、强调 |
| **缓推** | 渐强 | 紧张、聚焦 | 揭示、强调、悬念 |
| **缓拉** | 渐弱 | 释放、展开 | 揭示全局、结束 |
| **平移** | 流畅 | 跟随、展示 | 展示环境、跟随动作 |
| **跟随** | 动态 | 参与、沉浸 | 跟随角色、动作 |
| **环绕** | 强调 | 全面、重要 | 展示物品、重要时刻 |

## 画面构成参考

### 构图原则

| 构图 | 视觉效果 | 适用场景 | Veo3 描述方式 |
|------|---------|---------|--------------|
| **中心构图** | 强调、正式 | 单一主体、重要物品 | `centered composition`, `subject in center` |
| **三分法** | 平衡、自然 | 大多数场景 | `rule of thirds`, `off-center placement` |
| **对称** | 稳定、对比 | 对比场景、正式场合 | `symmetrical composition`, `mirrored layout` |
| **对角线** | 动态、张力 | 动作场景、紧张感 | `diagonal composition`, `dynamic angles` |
| **框架构图** | 聚焦、层次 | 突出主体、增加深度 | `framed by [element]`, `natural frame` |
| **负空间** | 简洁、高级 | 强调主体、留白美学 | `negative space`, `minimalist composition` |

### 画面层次设计

```
前景 (Foreground)
├── 作用：增加深度、引导视线、框架构图
├── 常用元素：虚化物品、装饰元素、引导线
└── Veo3: "in the foreground...", "foreground elements include..."

中景 (Midground) - 视觉焦点
├── 作用：主要内容、视觉焦点、信息传递
├── 常用元素：主体角色、关键物品、主要动作
└── Veo3: "in the center...", "the main subject..."

背景 (Background)
├── 作用：建立环境、营造氛围、提供语境
├── 常用元素：环境设定、氛围元素、色调基调
└── Veo3: "against a backdrop of...", "background shows..."
```

### 常用画面构成模式

#### 1. 单主体构成
```
[   背景环境   ]
[      🎯      ]  ← 主体居中或三分法位置
[   前景元素   ]
```
**适用**：产品展示、角色特写、重要物品

#### 2. 双元素对比
```
[      背景      ]
[  🅰️    vs    🅱️  ]  ← 左右对称或对角
[      前景      ]
```
**适用**：对比、选择、前后对照

#### 3. 多元素环绕
```
[      背景      ]
[  🔵  🎯  🟢  ]  ← 元素围绕中心
[      🟡      ]
```
**适用**：列举、分类、选项展示

#### 4. 流程展示
```
[      背景      ]
[ 1️⃣ → 2️⃣ → 3️⃣ ]  ← 横向或纵向流程
[      前景      ]
```
**适用**：步骤、流程、变化过程

#### 5. 场景叙事
```
[      远景环境      ]
[  角色    物品    目标  ]  ← 叙事元素分布
[      前景装饰      ]
```
**适用**：故事场景、情境展示

## 动态设计模式

### 物品动态库

| 动态类型 | 描述 | Veo3 关键词 | 适用场景 |
|---------|------|------------|---------|
| **飞入** | 物品从画面外飞入 | `flies in from [direction]`, `enters frame` | 出现、加入 |
| **飞出** | 物品飞出画面 | `flies away`, `exits frame`, `departs` | 离开、消失 |
| **弹出** | 物品弹跳出现 | `pops up`, `bounces into view`, `springs up` | 惊喜、强调 |
| **淡入** | 物品渐渐显现 | `fades in`, `gradually appears`, `materializes` | 柔和出现 |
| **旋转** | 物品自转展示 | `rotates`, `spins`, `turns slowly` | 展示、强调 |
| **缩放** | 物品大小变化 | `grows larger`, `shrinks down`, `scales up` | 强调、变化 |
| **变形** | 物品形态改变 | `transforms into`, `morphs`, `shape-shifts` | 转变、魔法 |
| **分裂** | 物品一变多 | `splits into multiple`, `duplicates`, `multiplies` | 复制、增多 |
| **合并** | 多物品合一 | `merge together`, `combine into`, `fuse` | 整合、连接 |
| **漂浮** | 物品悬浮移动 | `floats`, `hovers`, `drifts gently` | 梦幻、轻盈 |
| **坠落** | 物品下落 | `falls down`, `drops`, `descends` | 失去、结束 |
| **弹跳** | 物品反复跳动 | `bounces`, `hops`, `springs` | 活泼、强调 |

### 场景动态库

| 动态类型 | 描述 | Veo3 关键词 | 适用场景 |
|---------|------|------------|---------|
| **光线变化** | 明暗变化 | `lighting shifts from [A] to [B]`, `illumination changes` | 情绪转换 |
| **色调变化** | 颜色氛围改变 | `color palette transitions`, `hue shifts` | 情感变化 |
| **背景切换** | 背景场景更换 | `background transforms`, `scene shifts` | 场景转换 |
| **天气变化** | 天气效果 | `weather changes`, `rain begins`, `clouds part` | 氛围营造 |
| **时间流逝** | 时间推移效果 | `time passes`, `day turns to night` | 时间概念 |
| **空间扩展** | 空间变大 | `space expands`, `room grows larger` | 开阔、可能性 |
| **聚光效果** | 聚光灯照射 | `spotlight illuminates`, `light focuses on` | 强调、揭示 |

### 角色动态库（如需要角色）

| 动态类型 | 描述 | Veo3 关键词 |
|---------|------|------------|
| **表情变化** | 面部表情改变 | `expression changes from [A] to [B]`, `face transforms` |
| **肢体动作** | 手臂/身体动作 | `reaches out`, `gestures`, `points at`, `waves` |
| **位移移动** | 角色移动 | `walks toward`, `steps back`, `moves across` |
| **互动操作** | 与物品互动 | `picks up`, `places down`, `opens`, `touches` |
| **反应动作** | 对事件的反应 | `reacts with surprise`, `responds by`, `notices and` |

## 视觉隐喻设计原则

### 好的视觉隐喻特征

1. **即时可读性**：观众立即理解含义
2. **美感**：视觉上令人愉悦
3. **独特性**：与众不同，令人记忆
4. **一致性**：与整体风格协调
5. **简洁性**：不过度复杂

### 视觉隐喻设计流程

```
1. 提取核心概念
   └── 这句话本质在说什么？

2. 寻找具象对应
   └── 什么物品/场景能代表这个概念？

3. 设计动态表达
   └── 如何用动作/变化来强化表达？

4. 检查可读性
   └── 观众能在2秒内理解吗？

5. 确保美感
   └── 画面是否令人愉悦？
```

### 隐喻设计技巧

| 技巧 | 描述 | 示例 |
|------|------|------|
| **直接替换** | 用具象物替代抽象概念 | 时间 → 沙漏、钟表 |
| **拟人化** | 赋予物品人的特征 | 钱包露出悲伤表情 |
| **夸张放大** | 放大某个特征 | 省下的钱 → 金币雨 |
| **对比并置** | 将对立元素放在一起 | 昂贵课程 vs 免费资源 |
| **变形转化** | 物品A变成物品B | 问号变成灯泡 |
| **因果链条** | 展示原因和结果 | 点击按钮 → 卡片飞出 |

## 风格一致性指南

### 保持一致的元素

| 元素 | 说明 |
|------|------|
| **材质风格** | 整个视频使用同一种风格（黏土/毛毡/像素/霓虹/水彩） |
| **色彩调性** | 保持统一的色彩风格和饱和度 |
| **角色造型** | 同一角色在不同场景中外观一致 |
| **光影风格** | 统一的光源方向和色温 |
| **动态节奏** | 相似的动画速度和缓动曲线 |

### 各风格的一致性要点

| 风格 | 必须保持一致的特征 |
|------|------------------|
| **Style A 黏土风** | 光滑表面、指纹纹理、饱和色彩、硬边阴影 |
| **Style B 毛毡风** | 毛绒纤维、柔和边缘、暖色调、软阴影 |
| **Style C 像素风** | 像素网格大小、抖动技法、调色板限制、锯齿边缘 |
| **Style D 霓虹风** | 紫粉色调、霓虹辉光、冷暖对比、细节密度 |
| **Style E 水彩风** | 纸纹理、晕染边缘、留白、透明叠色 |

### 场景间的连贯性

- **视觉元素延续**：某些道具或元素在多个场景中出现
- **色彩过渡**：相邻场景的主色调有关联
- **动作衔接**：前一场景的结束动作与下一场景开始呼应
- **节奏统一**：整体保持相似的节奏感

## Veo3 优化技巧

### 提示词优化

1. **具体优于抽象**
   - ❌ `a character moves`
   - ✅ `a small orange felted fox reaches out with tiny paws`

2. **动态描述要完整**
   - ❌ `the box opens`
   - ✅ `the ornate treasure chest slowly creaks open, golden light bursts outward`

3. **指定材质和质感**
   - ❌ `clay style`
   - ✅ `needle felting style with visible wool fibers and fuzzy texture`

4. **明确镜头语言**
   - ❌ `camera moves`
   - ✅ `slow dolly forward, camera pushes in toward the glowing treasure`

5. **包含情感氛围**
   - ❌ `nice lighting`
   - ✅ `warm golden hour lighting creating cozy inviting atmosphere`

### 避免的问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 动作模糊 | Veo3 不知道具体怎么动 | 描述起始、过程、结束 |
| 风格不明确 | 默认生成写实风格 | 开头明确声明风格类型 |
| 镜头混乱 | 多种镜头混在一起 | 每个场景一种主要镜头 |
| 过于复杂 | 单个提示词内容过多 | 简化，一个场景一个核心动作 |
| 风格混淆 | 像素/水彩/霓虹特征不明显 | 强调风格核心特征词 |

### 各风格的常见问题与解决

| 风格 | 常见问题 | 解决方案 |
|------|---------|---------|
| **像素风** | 边缘过于平滑 | 强调 `visible pixel grid`, `aliased stepped edges` |
| **霓虹风** | 色彩不够浓烈 | 强调 `neon purple magenta`, `glowing`, `high contrast` |
| **水彩风** | 变成数字绘画 | 强调 `paper texture`, `pigment bleeding`, `watercolor washes` |
