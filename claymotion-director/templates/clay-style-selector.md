# 黏土动画风格选择器 (Clay Style Selector)

## 用途
定义两种核心黏土动画风格的视觉锚点、关键词和技术参数，用于后续提示词生成。

---

## Style A: Tech-Polymer (硬核光面黏土)

### 视觉锚点
- **质感**: 类似 "Polymer Clay" (聚合物黏土) 的光面质感
- **表面**: 轻微反光、光滑但保留手工捏制痕迹
- **色彩**: 饱和度高、偏向科技感的配色 (深蓝/霓虹绿/紫红/银灰)
- **光影**: 定向强光源 + 环境光反射，营造"科技实验室"氛围

### 关键词库
```
材质关键词:
- polymer clay, glossy clay,光滑黏土, 光面聚合物
- 捏痕可见, 指纹细节, 手工痕迹
- 半透明边缘, 局部反光

场景关键词:
- 科技实验室背景, 金属桌面, LED 灯带
- 几何形状底座, 网格地面
- 蓝色/紫色 环境光

动作风格:
- 机械感动作, 弹性变形, 快速移动
- 零件组装/拆卸, 数据流动, 能量传输
```

### 技术参数
| 参数 | 值 |
|------|-----|
| 光照 | 定向主光 + 蓝紫色轮廓光 |
| 阴影 | 硬阴影，边缘清晰 |
| 反射 | 轻微镜面反射 |
| 背景 | 简洁科技风 |

---

## Style B: Cozy-Felt (暖心羊毛毡)

### 视觉锚点
- **质感**: 类似 "Needle Felting" (羊毛毡) 的毛绒质感
- **表面**: 毛绒绒、纤维可见、温暖柔软
- **色彩**: 低饱和暖色调 (米黄/粉橙/草绿/天蓝)
- **光影**: 柔和漫射光 + 暖色调，营造"温馨手作"氛围

### 关键词库
```
材质关键词:
- needle felted, wool felt, 羊毛毡, 毛绒材质
- 纤维蓬松, 毛边可见, 针戳痕迹
- 柔软变形, 轻微毛躁感

场景关键词:
- 木纹桌面, 温暖家居背景
- 棉线/布料道具, 手工感装饰
- 暖黄色/橙色 环境光

动作风格:
- 缓慢弹性移动, Q弹变形
- 拥抱/依偎/传递, 温暖互动
- 手工缝制/编织动作
```

### 技术参数
| 参数 | 值 |
|------|-----|
| 光照 | 柔和漫射光 + 暖色补光 |
| 阴影 | 软阴影，边缘模糊 |
| 反射 | 无镜面反射（哑光） |
| 背景 | 温馨家居风 |

## 风格选择决策树

### 根据文案类型推荐

| 文案类型 | 推荐风格 | 理由 |
|---------|---------|------|
| 技术产品/架构/API | **Style A: Tech-Polymer** | 质感匹配"硬核科技"，适合表达精确、逻辑 |
| 商业模式/增长/运营 | **Style B: Cozy-Felt** | 质感匹配"温暖增长"，适合表达连接、协作 |
| 开发者工具/DevOps | **Style A: Tech-Polymer** | 适合表达"构建/组装/调试"的技术动作 |
| 用户故事/情感共鸣 | **Style B: Cozy-Felt** | 适合表达"关怀/陪伴/成长"的温暖情感 |
| 数据分析/机器学习 | **Style A: Tech-Polymer** | 适合表达"数据流动/模型训练"的抽象概念 |
| 教育/培训/知识分享 | **Style B: Cozy-Felt** | 适合表达"传递/启发"的教育场景 |

### 混合风格策略
当文案同时包含"技术"和"情感"元素时：
- **主风格**: 根据核心信息选择（技术干货 → A，情感故事 → B）
- **辅助元素**: 在次要角色/道具上使用对比风格，突出主次关系
