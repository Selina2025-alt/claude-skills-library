## A. 镜头角度选择

根据分镜内容选择最佳镜头角度。

### 镜头角度库

| 角度 | 视觉效果 | 适用场景 | Veo3 关键词 |
|------|---------|---------|------------|
| **正面** | 直接、亲密、对话感 | 情感表达、与观众对话、揭示表情 | `frontal view`, `facing camera` |
| **侧面** | 动态、过程、叙事感 | 动作展示、移动过程、场景切换 | `side view`, `profile shot` |
| **背面** | 神秘、展望、带入感 | 看向目标、引导视线、制造悬念 | `from behind`, `over shoulder` |
| **俯视** | 全局、掌控、信息展示 | 桌面场景、流程图、物品排列 | `top-down view`, `bird's eye`, `overhead shot` |
| **仰视** | 崇高、威严、力量感 | 权威展示、高大形象、敬畏感 | `low angle`, `looking up` |
| **特写** | 聚焦、强调、细节感 | 关键动作、重要物品、情绪细节 | `close-up`, `macro shot`, `detail shot` |
| **中景** | 平衡、自然、叙事感 | 角色互动、日常场景、对话 | `medium shot`, `waist shot` |
| **远景** | 环境、氛围、史诗感 | 场景建立、环境展示、开场收尾 | `wide shot`, `establishing shot` |
| **斜角** | 紧张、不安、动态感 | 冲突、紧张时刻、打破常规 | `dutch angle`, `tilted frame` |
| **环绕** | 360度展示、全面了解 | 产品展示、重要物品、全方位介绍 | `orbiting shot`, `rotating around` |

### 镜头选择决策树

```
文案内容是什么？
├─ 展示情感/表情 → 正面特写
├─ 展示动作/过程 → 侧面中景
├─ 展示全局/布局 → 俯视远景
├─ 展示物品/细节 → 特写/环绕
├─ 展示目标/未来 → 背面 + 目标物
├─ 展示权威/重要 → 仰视
└─ 展示对比/选择 → 正面 + 分屏构图
```

## B. 画面构成设计

设计画面中各元素的位置和关系。

### 画面构成模式

| 模式 | 描述 | 构成比例 | 适用场景 |
|------|------|---------|---------|
| **角色主导** | 角色是视觉中心 | 角色 60% + 环境 40% | 情感故事、角色表演 |
| **场景主导** | 环境是视觉中心 | 环境 70% + 角色/物品 30% | 氛围营造、场景建立 |
| **物品主导** | 特定物品是焦点 | 物品 60% + 背景 40% | 产品展示、关键道具 |
| **动态主导** | 动作/变化是焦点 | 动态过程占主体 | 流程演示、变化展示 |
| **并列构图** | 多元素平等呈现 | 2-4 个元素均分 | 对比、列举、选择 |
| **层次构图** | 前中后景分明 | 前景 20% + 中景 50% + 背景 30% | 丰富场景、深度感 |

### 画面元素清单

为每个分镜设计以下元素：

```markdown
**前景**: [最靠近镜头的元素，可用于构图引导或景深]
**中景**: [主要内容所在区域，视觉焦点]
**背景**: [环境氛围、场景建立]
**道具**: [辅助表达的物品]
**特效**: [光效、粒子、UI元素等]
```

### 构图参考

| 构图类型 | 描述 | 适用场景 |
|---------|------|---------|
| **中心构图** | 主体居中 | 强调单一主体、正式感 |
| **三分法** | 主体在 1/3 线交点 | 自然、平衡、专业 |
| **对称构图** | 左右/上下对称 | 对比、选择、平衡 |
| **引导线构图** | 线条引向主体 | 引导视线、动态感 |
| **框架构图** | 用前景元素框住主体 | 聚焦、层次感 |
| **负空间** | 大量留白 | 简洁、高级、强调 |

## C. 动态效果规划

设计画面中的运动和变化。

### 动态类型

| 动态类型 | 描述 | 示例 |
|---------|------|------|
| **角色动作** | 角色的肢体运动 | 挥手、转身、跳跃、指向 |
| **物品动态** | 物品的移动/变化 | 飞入、弹出、旋转、变形、复制 |
| **场景动态** | 环境的变化 | 光线变化、背景切换、天气变化 |
| **镜头动态** | 镜头的移动 | 推近、拉远、平移、跟随、环绕 |
| **UI动态** | 界面元素动画 | 数字跳动、进度条、图表生长 |
| **特效动态** | 视觉特效 | 粒子、光晕、闪烁、轨迹 |

### 动态设计模板

```markdown
**起始状态**: [画面初始样子]
**动态过程**: 
  - [时间点1]: [发生什么]
  - [时间点2]: [发生什么]
  - [时间点3]: [发生什么]
**结束状态**: [画面最终定格]
**镜头运动**: [镜头如何配合移动]
```

### 镜头运动库

| 运动类型 | 效果 | Veo3 关键词 |
|---------|------|------------|
| **静止** | 稳定、专注、传统定格感 | `static camera`, `locked off` |
| **推近** | 强调、聚焦、紧张感 | `push in`, `dolly forward`, `slow zoom in` |
| **拉远** | 揭示、展开、释放感 | `pull back`, `dolly out`, `zoom out revealing` |
| **平移** | 跟随、展示、叙事感 | `pan left/right`, `horizontal tracking` |
| **跟随** | 动态、陪伴、沉浸感 | `tracking shot`, `following` |
| **环绕** | 360度、全面、展示感 | `orbiting`, `rotating around`, `circular tracking` |
| **升降** | 揭示、俯瞰、转换视角 | `crane up/down`, `vertical movement` |
| **抖动** | 紧张、冲击、手持感 | `handheld`, `slight shake` |

### 转场方式

| 转场 | 效果 | Veo3 关键词 |
|------|------|------------|
| **硬切** | 快速、干脆、节奏感 | `hard cut`, `cut to` |
| **溶解** | 柔和、过渡、时间流逝 | `dissolve`, `crossfade` |
| **淡入淡出** | 开始/结束、段落分隔 | `fade in`, `fade out`, `fade to black` |
| **匹配剪辑** | 巧妙、连贯、创意感 | `match cut` |
| **擦除** | 方向性、推进感 | `wipe transition` |
| **缩放转场** | 聚焦、放大某元素 | `zoom transition` |

## 完整示例

### Scene 1: 开场引入

**分镜文案**: 都2026年了，求求你们别再花几千块去报AI速成班了
**视觉隐喻**: 钞票飞走 + 无奈感

### 镜头设计
- **角度**: 俯视 45度
- **景别**: 中景
- **镜头运动**: 静止 → 轻微推近

### 画面构成
- **构成模式**: 场景主导 + 物品动态
- **前景**: 散落的钞票（正在飞走）
- **中景**: 温暖的木质桌面，放着一个标有"AI速成班 ¥3999"的招牌
- **背景**: 柔和的暖色调家居环境
- **道具**: 钞票（飞走动画）、价格招牌、空钱包

### 动态设计
- **起始状态**: 桌面上钞票整齐摆放，钱包鼓鼓的
- **动态过程**: 
  - 钞票长出小翅膀，开始扑腾
  - 钞票一张张飞向"AI速成班"招牌
  - 钱包逐渐瘪下去
- **结束状态**: 钱包完全瘪掉，最后一张钞票飞走
- **特效**: 钞票飞走留下闪烁轨迹

### 角色设计
- **角色**: 无（本场景不需要角色）

### 转场
- **与下一场景的转场**: 溶解过渡

### Scene 2: 揭示免费渠道

**分镜文案**: 今天我把官方下场喂饭学AI的免费渠道扒出来

- **景别**: 中景 → 特写

### 画面构成

- **前景**: 无

- **背景**: 柔和的金色光芒从宝箱中透出
