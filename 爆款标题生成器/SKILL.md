## 🎯 六大核心策略

### 策略1：认知反差策略 ⭐⭐⭐⭐⭐

**原理**：打破用户预期，制造"以为是A，其实是B"的惊讶感

**高频词库**：
- 程度词：离谱、强到、太、终于、竟然
- 否定词：不是、再见了、告别、我摊牌了
- 对比词：单纯vs复杂、伪装、真相

**万能模版**：
```
[形容词]到[程度]？[主体]_[核心动作]！
太[形容词]了！[主体]_[疑问/感叹]
不是[常规认知]！[主体]_[真相]
终于[动词]！我用上[产品/工具]了！
```

**实战案例**：
- "强到离谱？Gemini 3深度实测"
- "AI很单纯，复杂的是人"
- "不是程序员才需要用cursor！"
- "终于等到！我用上SORA了！"

---

### 策略2：数字暴击策略 ⭐⭐⭐⭐⭐

**原理**：用具体数字降低认知成本，制造可信度和价值感

**数字类型**：
- **时间类**：4分钟、30秒、5分钟、1天、3小时
- **数量类**：300回合、60集、6大秘籍、4招、N个理由
- **金额类**：2000块、10w奖金、月入2万
- **步骤类**：简单4步、3步、1分钟

**万能模版**：
```
[数字][时间/步骤]，[主体]_[核心动作]！
[数字][时间]全了解！[主体]_[标签]
实测[数字]次！[主体]_[结果]
[数字]步搞定！[主体]_[小白/神器标签]
```

**实战案例**：
- "4分钟教你用AI生成自己的脸！"
- "300回合狂测GPT5，有点真东西"
- "简单4步，带你上手微调大模型"
- "1天节省3小时，用AI给'信息降噪'！"

## 🛠️ 标题组合公式

### 黄金三段式公式

```
[情绪钩子] + [核心价值] + [行动标签]
```

**示例拆解**：
- "强到离谱？Gemini 3深度实测【玩法合集】"
  - 情绪钩子：强到离谱？（认知反差）
  - 核心价值：Gemini 3深度实测（产品+动作）
  - 行动标签：【玩法合集】（内容承诺）

- "小白也能30秒做出网站【最新神器v0.dev】"
  - 情绪钩子：小白也能30秒（降低门槛）
  - 核心价值：做出网站（明确结果）
  - 行动标签：【最新神器v0.dev】（工具推荐）

## 📝 标题自检清单

一个合格的爆款标题应该包含：

- [ ] 至少1个具体数字
- [ ] 明确的目标受众（小白/专业/特定人群）
- [ ] 清晰的价值承诺
- [ ] 情绪化标点（！？～）
- [ ] 行动导向动词
- [ ] 可选：标签【】强化属性
