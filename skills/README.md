# Claude Code Skills 索引

这是我的 Claude Code 技能库，包含各种自定义技能。

## 📋 技能列表

### 帮我写作
结合用户提供的材料进行文案写作，特别适合软件/AI/计算机相关内容的深度文案创作。

**使用方法**：提供材料后，skill 会自动学习你的行文风格（基于 references/ 中的范文）并撰写流畅、有节奏感的口播文案。

### 来点选题
当视频创作选题枯竭时，提供 10-15 个类似的灵感选题。

**使用方法**：基于你往期数据较好的视频分析，生成符合你风格的选题建议。

### 英文播客自动总结
一键完成播客的抓取、转录和智能分析。

**使用方法**：
```bash
/podcast-analyze
```

自动执行：RSS 抓取 → Deepgram 转录 → Claude 分析 → 保存结果全流程。

### 字幕转 markdown
将 SRT 字幕文件转换成 markdown 笔记格式。

**使用前提**：需要先安装 ffmpeg 并放入 `C:\Windows\System32`

---

## 🔄 新增技能流程

每当创建新 skill 时，按以下步骤添加到此仓库：

1. 复制 skill 目录到 `skills/` 文件夹
2. 在此文件中添加技能说明
3. 提交到 Git 仓库
